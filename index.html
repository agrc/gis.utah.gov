Spatial Data

Established by state law in 1991, Utah’s State Geographic Information Database (SGID) provides one-stop access to hundreds of GIS map data layers developed, aggregated, or acquired by state government. Roads, boundaries, addressing, parcels, aerials, elevation, etc...you’ll find it available for download and as web services from the SGID.

Geo APIs

Does your application just need a quick answer to a simple “Where is?” or “What’s At?” question? Many basic location needs can be integrated into your application using the address geocoding and spatial point-in-polygon queries available through api.mapserv.utah.gov.

Aerial Imagery & Base Maps

SGID aerial photography and base map services efficiently provide critical map context for GIS and CAD users, as well as web and mobile applications. Multiple aerial photography products and vintages are available for download and use as web services. Several base map themes depict data sourced from state, county, and other local data sources.

TURN GPS Network

TURN GPS is the foundation for live, high precision GPS field surveying, mapping, and other measurement. TURN GPS is a statewide network of over 90 permanently located GPS receivers, that enables sub-centimeter level precision for Utah, and portions of surrounding states including southern Idaho, Western Wyoming, and the Las Vegas metro area.

Application Development

AGRC’s development team builds and hosts web applications and services designed to best meet agency/program requirements. Tailored web applications can help an organizations to more effectively gather, manage, and communicate the information that is most critical to their mission.

GIS Map Data

Utah's State Geographic Information Database (a.ka. "the SGID"), was established by policy makers to ensure that GIS map layers were developed in a coordinated fashion and shared openly, without redundant effort. The vision for the SGID was created almost two full decades before the govenrment 'open data' and 'transparency' initiatives started hitting their stride. And, to the credit of the GIS community across the state, the SGID's GIS data resources are among the most mature and extensive in the nation.

Today, interoperable GIS server technology has diminished the importance of a completely centralized map data repository. But, it's actually more critical that the latest, best quality GIS data resources are readily discovered and accessed. And, that's true for resources available to users as downloadable shapefiles, as well as, resources available as connections to live streaming data services, AGRC strives to ensure that the SGID stays current as a state-level channel to aide in the discovery, access, and use of GIS data resources.

Base Maps and Imagery

AGRC hosts aerial photography and pre-rendered base map services on its discover.agrc.utah.gov cloud-based server. After obtaining an organizational login, GIS users, CAD users, and web apps can connect to these multi-scale maps using the Open Geospatial Consortium WMS and WMTS protocols. The Discover server provides images and tiles in the Web Mercator projection, an industry standard for web map display.

Layers accessible from the Discover server include themed base maps built from local & state GIS data in terrain, lite, topo, and hybrid themes. Aerial photography offerings include the latest 6" statewide imagery licensed for state, local, & tribal government from Google, as well as public domain color NAIP, Wasatch Front HRO, CIR/NRG imagery, and (coming soon) historic B/W DOQ series.

Through the end of 2016, AGRC also provides basemaps from its mapserv.utah.gov server in UTM NAD83 coordinates. Though these base map services are still available are expected to be retired in favor of the cloud-hosted OGC services served through Discover, map refreshes to mapserv base map services will be limited.

TURN GPS Precision RTK Network

The Utah Reference Network for GPS (TURN GPS) is a networked system of over 90 permanently located high precision GPS receivers, installed across Utah and neighboring states.

TURN GPS provides a real-time correction signal that factors out atmospheric variation between earth-orbiting GPS satellites and ground-based GPS receivers in the field.

TURN GPS Network Coverage

Surveyors, construction crews, field mapping technicians, and GPS-controlled machinery are able to operate with up-to sub-centimeter precise measurement by connecting their existing GPS receivers to the TURN GPS correction service over a smart phone hotspot.

A partnership with state and local government agencies has allow TURN GPS base stations to be sited at existing Utah facilities in exchange for the partners' use of the system. One year subscriptions to the TURN GPS service are $600 per GPS device.

APIs and JavaScript Widgets

APIs available through api.mapserv.utah.gov enable common tasks like address location (aka geocoding) and map search (point-in-polygon query) to be built into web applications and data-handling processes without the immediate need for GIS-specific software. These API's are at the core of sites like Who Represents Me?, Who Taxes me? etc.

AGRC's dev group maintains a library of Utah-specific map widgets that provide common tools desired in web mapping applications. The library includes FindAddress (geocoding), TRSSearch (finds PLSS sections), and Sherlock (can be used to search for any feature in the SGID or layers in a map services. We also, on occasion publish customization for ArcGIS desktop like our geocoding api tool.

We are into git. By default, the AGRC dev team keeps its code current in AGRC's GitHub repository. If it's a web site, widget, desktop add-in, or even the content of this website itself, it should be findable, and open-sourced in the AGRC repo.
