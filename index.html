Data

Established in state law in 1991, Utah’s State Geographic Information Database (SGID) provides one-stop access to hundreds of GIS map data layers developed, aggregated, or acquired by state government. Roads, boundaries, addressing, parcels, aerials, elevation, etc...you’ll find it available for download and as web services from the SGID.

Geo APIs

Does your application just need a quick answer to a “Where is?” or “What’s At?” Many basic location needs can be integrated into your application using the address geocoding and spatial point-in-polygon queries available through api.mapserv.utah.gov.

Aerial Imagery & Base Maps

SGID aerial imagery photography and base map services efficiently provide critical map context for GIS and CAD users, as well as web and mobile applications. Multiple aerial photography products and vintages are available for download and use as web services. Several base map themes are all sourced from state and local data sources.

TURN GPS Network

TURN GPS is the foundation for live, high precision GPS field surveying and field measurement. TURN GPS is a statewide network of over 90 permanently located GPS receivers, that enables sub-centimeter level precision for Utah, and portions of surrounding states including southern Idaho, Western Wyoming, and the Las Vegas metro area.

Application Development

AGRC’s development team builds and hosts web applications and services designed to best meet business requirements. Tailored web applications can help an organizations to more effectively gather, manage, and communicate the information that is most critical to their mission.

Coordination and information sharing, in the form of news, events, tips, and data, fuels our collective success.

GIS Map Data

Utah's State Geographic Information Database (a.ka. "the SGID"), was established by policy makers to ensure that GIS map layers were developed in a coordinated fashion and shared openly, without redundant effort. The vision for the SGID was created almost two full decades before the govenrment 'open data' and 'transparency' initiatives started hitting their stride. And, to the credit of the GIS community across the state, the SGID's GIS data resources are among the most mature and extensive in the nation.

Today, interoperable GIS server technology has diminished the importance of a completely centralized map data repository. But, it's actually more critical that the latest, best quality GIS data resources are readily discovered and accessed. Whether these resources are available to users as downloadable shapefiles or as connections to live streaming data services, AGRC strives to ensure that the SGID stays current as a state-level channel to aide in the discovery, access, and use of GIS data resources.

Base maps and Imagery

AGRC hosts aerial photography and pre-rendered base map services on its discover.agrc.utah.gov cloud-based server. GIS users, CAD users, and web apps can connect to these mutli-scale maps using the Open Geospatial Consoortium WMS and WMTS protocols. The discover server provides images and tiles in the Web Mercator projection, an industry standard for web map display.

Layers accessible from discover server include themed base maps built from local & state GIS data in terrain, 'lite', topo, and hybrid themes. Aerial photography includes the latest 6" statewide imagery licensed for state, local, & tribal government from Google, as well as public domain color NAIP, CIR/NRG imagery, and (coming soon) historic B/W DOQ series.

Precision RTK GPS Network

The Utah Reference Network for GPS (TURN GPS) is a networked system of over 90 permanently located high precision GPS receivers, installed across Utah and neighboring states.

TURN GPS provides a real-time correction signal that factors out atmospheric variation between earth-orbiting GPS satellites and ground-based GPS receivers in the field.

TURN GPS Network Coverage

Surveyors, construction crews, field mapping technicians, and GPS controlled machinery are able to operate with sub-centimeter precise measurement by connecting their existing GPS receivers to the TURN GPS correction service over a smart phone data plan.

A partnership with state and local government agencies has allow TURN GPS base stations to be sited at existing Utah facilities in exchange for use of the system. One year subscriptions to the TURN GPS service are $600 per GPS device.

APIs and JavaScript Widgets

APIs available through api.mapserv.utah.gov enable common tasks like address location (aka geocoding) and map search (point-in-polygon query) to be built into web applications and data-handling processes without the immediate need for GIS-specific software.

AGRC's dev group maintains a library of Utah-specific map widgets that provide common tools desired in web mapping applications. The widgets include FindAddress, FindTRS (zoom to PLSS sections), and FindGeneric (can be used to search for any feature in an SGID layer. We also, on occasion publish customization for ArcGIS desktop like our geocoding api tool.

We are into git. By default, the AGRC dev team keeps its code current in AGRC's github repository. If it's a web site, widget, desktop add-in, or even the content of this website itself, it should be findable, and open-sourced in the AGRC repo.
