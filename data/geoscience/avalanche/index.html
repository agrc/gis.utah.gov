Avalanche Paths

Data Type: GIS Data Layer
Steward: AGRC

Abstract:
This dataset represents snow avalanche paths in the tri-canyon area of the Wasatch Front, Utah.
This dataset also contains the name, size of slidepath, return interval, starting zone elevation, vertical fall and starting zone aspect for some of the avalanches.
This data was prepared for the Salt Lake County Wasatch Canyons Master Plan originally published in September 1989.

Usage: There are no constraints or warranties with regard to the use of this dataset.
