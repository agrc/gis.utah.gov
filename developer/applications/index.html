Watershed Restoration Initiative Web Application AGRC, DNR, and DWR The Watershed Restoration Initiative website uses a shared web map to enable interagency project and resource coordination. Vehicle Collision Map Web Application AGRC, DPS, and UDOT A statewide map of vehicle crashes, reported across all jurisdictions, that allows dynamic filtering for time, date, cause, severity, etc. Economic Development Map Web Application AGRC and GOED Utah's Economic Development Map depicts the landscape of what's important to businesses considering relocating to Utah. Environmental Interactice map Web Application AGRC and DEQ Search for site-specific information relating to DEQ programs including regulated environmental sites, facilities, and designations. Raster Data Discovery Web Application AGRC Download all of the imagery, elevation, lidar, and scanned maps available from Utah's SGID. State Parcel Map Web Application AGRC Zoom in to view parcel boundaries and basic attributes as assembled statewide by AGRC, in coordination with County Recorders.
