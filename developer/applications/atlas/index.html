Atlas

Tools for visualizing Utah Roads, Imagery, and Widgets

external link atlas.utah.gov github repository link Repository download data link Download Application Data

Why

UGRC offers many online services to facilitate the use of GIS and Atlas is a combination of most of those services. Atlas contains the complete and most current Discover base maps. Atlas also uses most of the UGRC Components. Clicking anywhere on the map makes http requests to the UGRC Web API to reverse geocode, search for the city, county, land ownership, elevation, and US National Grid of the point clicked; and display the results in the left panel. For UGRC developers, Atlas is a code base to keep improving its usage of dojo and the ArcGIS Maps SDK for JavaScript. Atlas is a GitHub template that you can use to create your own website.
