Atlas

Tools for visualizing Utah Roads, Imagery, and Widgets

atlas.utah.gov external link github repository link Repository download data link Download Application Data

Why

AGRC offers many online services to facilitate the use of GIS and Atlas is a combination of most of those services. Atlas contains the complete and most current discover base maps. Atlas also uses most of the AGRC javascript widgets. The popup makes http requests to the AGRC Web API to reverse geocode, search for the city, county, land ownership, elevation, and US National Grid of the point clicked. For AGRC developers, Atlas is a code base to keep improving its' usage of dojo and the esri js api. Atlas is often cloned and used as an application boilerplate for new projects as it's structure is optimal for the dojo build process.
